# Proyecto Final React CoderHouse

## Clothing Stores
Clothing store es una pagina eCommerce dise単ada en react como proyecto final de curso de CorderHouse.
***
## Caracteristicas 
Obtiene los productos de la base de datos de Firebase y los muestra al cliente. Este mismo al terminar la compra envia sus datos y sus productos comprados a la misma base de datos. La pagina tiene filtros por categorias para la mayor comodidad en la busquedad de los productos. Se puede agregar y quitar productos del carrito y vaciarlo si es necesario. Se muestra la cantidad de productos elejidos en total en el Icono del Carrito en la NavBar y en cada producto elejido se muestra la cantidad de solo ese producto.
***
## Librerias Externas
Se utilizaron las siguientes librerias:
* React Router Dom: Se utilizo esta API para el manejo de rutas y links en la pagina dise単ada en SPA(single page aplicattion). Version: 6.3.0.
* React Hook Form : Se utilizo esta API para crear formularios ya que minimiza la cantidad de renderizaciones, el calculo de validacion y un montaje mas rapido. Ademas que es una peque単a biblioteca sin dependencias. Version: 7.31.1.
* Material UI: Se utilizaron sus componentes y su CSS incluido para el dise単o de la pagina.Tiene incluidos los iconos utilizados a los largo de la pagina. Es muy comoda su utilidad con React y es por eso que se elijio esta libreria. Version: 5.6.3.
* Firebase: Este proyecto esta concetado a la base de datos de Firebase v9.7.0. Ademas que nos brinda un hosting muy rapido.
